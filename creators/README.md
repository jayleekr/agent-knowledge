# 🎨 Creator Contributions

> HypeProof Lab Creator들의 지식 기여 공간.

## 참여 방법

1. `creators/{your-name}/` 디렉토리 생성
2. 교훈, 팁, 워크플로우 등 자유롭게 작성
3. PR로 제출 → 리뷰 후 머지

## 예시

```
creators/
├── jay/
│   ├── writing-tips.md
│   └── geo-optimization-notes.md
├── kiwon/
│   └── research-workflow.md
└── tj/
    └── tutorial-structure.md
```

## 규칙

- 마크다운 형식
- 개인정보 포함 금지
- 다른 Creator 디렉토리 수정 금지
- 유용한 것만! (일기장 아님)
